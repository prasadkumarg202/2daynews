04 · Geo-fencing — the heart

Language + region isolation, national shared

A GPS point resolves to a region via PostGIS point-in-polygon; the feed then returns local news for that language+region plus everything flagged national, and hides other states' local content.

  • Telugu → AP + Telangana + Tollywood; hides TN/KA local.
  • Hindi → follows GPS (Lucknow → UP; Bhopal → MP).
  • National → PM, SC, ISRO, UPSC, budget — shown to all.
  • Metro tie-break → is_metro desc when polygons overlap.
-- resolve GPS → region
create function region_for_point(lng, lat)
returns uuid as $$
  select id from regions
  where st_covers(geom,
    st_makepoint(lng,lat))
  order by is_metro desc limit 1;
$$;

-- the feed (geo-fenced)
select * from articles a
where status='published' and (
  a.is_national                -- shared
  or (a.language_id=$lang
      and a.region_id=$region))
order by is_breaking desc,
         priority desc,
         published_at desc;

06 · Cache, queues & model routing

Redis in front, BullMQ behind, four models blended

Redis cache

Feed responses cached per feed:<lang>:<region>:<section>.

  • Feed keys 30–60s TTL · live cricket 8s · gold rates 9m
  • Target hit rate ≥ 94% — misses fall through to get_feed
  • Publish event invalidates that lang+region key set
  • Also backs sessions, rate limits, poll-vote counters & BullMQ

BullMQ job queues

  • ingest-rss — poll & parse active sources
  • ai-rewrite — model calls, retry w/ backoff
  • push-fanout — breaking alerts to push_tokens
  • thumb-gen · social-syndicate
  • Failures → dead-letter + admin retry button

Model strategy

Gemini-only today — one structured call does rewrite + classify + fake-score (≈₹0.09/story).

  • Gemini Flash — all rewrite · classify · SEO · push copy
  • Dedup collapses N publishers → 1 Gemini call
  • Roadmap: Claude fact-check tier, GPT/DeepSeek fallbacks

Breaking desk — manual publish

Editors paste a raw tip / wire copy / URL in admin. POST /functions/v1/breaking runs the same pipeline as RSS: dedup → fact-check → fake-score gate → Gemini rewrite → SEO + push copy → publish with is_breaking=true, priority=10 → enqueue push-fanout. fake_score > 0.5 still routes to review — manual entry never bypasses the gate.

Engagement schema — polls, quizzes, reactions

  • polls — kind (poll|quiz|survey), question, options jsonb, category, language_id, region_id, placement, status, closes_at
  • poll_votes — poll_id, user_id, option_ix · RLS: own votes
  • reactions — article_id, user_id, type (like|share|save)
  • Feed interleaves live polls by category + placement rule
  • Vote counts on Redis counters, flushed to Postgres async
07 · Scale, security & cost

Built to reach 50M, priced to survive

Scale to 50M

Reads dominate 1000:1 — so cache hard and index for the feed.

  • CDN-cache get_feed at Cloudflare edge per language+region (30–60s TTL)
  • Composite index on (language_id, region_id, category, published_at)
  • Read replicas for feed; primary for pipeline writes
  • Archive stage moves cold rows out of hot tables

Security

  • RLS on users/bookmarks/follows/engagement/subs
  • Published articles world-read; writes service-role only
  • Gemini + service keys in Supabase secrets, never client
  • Phone-OTP auth (Supabase Auth); anon key for reads
  • Fake-news gate before publish; source trust weighting

Cost strategy

  • Free: Google News RSS, RSSHub, GDELT, govt RSS
  • Only per-article cost: Gemini (one call/story)
  • Dedup collapses N publishers → 1 AI call
  • Paid later: MediaStack/NewsAPI for latency-critical desks
  • Monetize: AdSense, sponsored, Plus tier, e-paper
