One-line: Language/script selector pill — the top-level control that geo-fences the entire feed; each pill shows a native script glyph and the language's own name.

```jsx
<LanguagePill code="te" active />
<LanguagePill code="ta" />
<LanguagePill code="hi" />
```

Built-in codes `te|ta|kn|ml|hi|en` carry native labels. `active` = ink fill. Override with `native`/`glyph`.
